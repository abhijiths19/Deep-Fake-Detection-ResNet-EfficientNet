Deepfake Detection

About the Project

This Deepfake Detection tool uses an ensemble of advanced deep learning models — ResNet50 and EfficientNetB0 — to analyze facial features in images and videos. By processing individual frames and detecting patterns commonly found in AI-generated content, this system classifies faces as either Real or Fake.

The aim is to combat the rising threat of deepfakes by providing a reliable detection platform that is fast, accurate, and easy to use. Upload a video or image, and let the model analyze the content in real time.

Built using Python, Flask, TensorFlow, OpenCV, and modern UI design principles.
